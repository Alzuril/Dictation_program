# Dictation_program
Program to write dictation in different foreign languages

This program is fully workable, but some of its fitches are not polished enough.

What this program can do:
	- add new words and divide them into topics
	- delete topics you do not need
	- show your wocabulary
	- give you 3 types of dictations such as:
		- dictation with self-check
		- dictation with keyboard input and autocheck
		- game in witch you have to choose 1 correct answer
